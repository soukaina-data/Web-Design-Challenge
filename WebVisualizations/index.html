    Summary : latitude vs.X

    The purpose of this projet was to analyse how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatheMap API to assemble a data over 500 cities.

    After assembling the dataset, we used Matlop tp plot various aspect of weather vs latitude. Factors we looked at included:Tempretures , cloudiness , wind speed, and humidity. This site provides the source data and visuslizations created as part of the analysis , as well as explanations and descriptions of any trends and correlations witnessed .
